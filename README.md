## Description

Compose-Learnings is a repository focused on learning **Jetpack Compose** for Android. The main showcases in this repository are Compose UI and Animations, using Jetpack Compose's modern toolkit for building native UI in Android. Additionally, Compose Navigation is used for navigation within the application.

### Technologies Used:

- **Jetpack Compose**
- **Compose Navigation**

Compose-Learnings is designed to provide examples and learning resources for building user interfaces with Jetpack Compose and animations in Android applications.
